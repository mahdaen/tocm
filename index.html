Tocm Class

Tocm Class is the core function of Tocm. With Tocm Class you can create and manage css easily and on the fly. $class(string name, object properties, string media); Sample: 

                $class('.header', {
                    display: 'block',
                    height: 400,
                    background_image: linear_gradient('180deg, #fff, #ccc')
                });

On the sample above, we're creating a css with .header as css selector and display, height, etc is a property. We're also can create same class but in different media type. To do this, we have to register the media query first to use when we create a class in a specific media. But we doesn't need to register again if we've already register this media. $media(string name, string rules); Sample: 

                $media('Desktop', 'screen and (min-width: 1024px)');

                $class('.header', {
                    display: 'block',
                    height: 400,
                    background_image: linear_gradient('180deg, #fff, #ccc')
                }, 'Desktop');

You can also use in-name identifier to specfy wich media will be used in the class. Look at this sample: 

                $class('.header@Desktop', {
                    display: 'block',
                    height: 400,
                    background_image: linear_gradient('180deg, #fff, #ccc')
                });

Both sample above is creating class for media 'Desktop'. By default, each class is writed in family-group node. So, if you want to create some class for Global Group, you have to specify the global identifier ('!') before the class name. Look at this sample: 

                $class('!.header@Desktop', {
                    display: 'block',
                    height: 400,
                    background_image: linear_gradient('180deg, #fff, #ccc')
                });

To create a class tree, simply do like this: 

                $class('.header', {
                    display: 'block',
                    height: 400,
                    background_image: linear_gradient('180deg, #fff, #ccc'),
            
                    '.logo': {
                        max_width: '100%'
                    },
                    '.menu': {
                        padding: 10, text_decoration: 'none', font_family: 'Tahoma',
                        
                        hover: {
                            color: '#444'
                        }
                    }
                });

Sample above is a standard method. It's like when we write .header .logo {} and .header .menu {} selector in css. Besides using standard method, you can aslo use chaining method. This method is generally used when modifying the class. Look at this sample: 

                $class('.header', {
                    display: 'block',
                    height: 400,
                    background_image: linear_gradient('180deg, #fff, #ccc')
                })
                .add('.logo', {width: '100%'})
                .add('.menu', {display: 'inline-block'});

To select the existing class, you can simply enter the name like this: 

                // Select universal class.
                $class('.header');
                // Select media class.
                $class('.header', 'Desktop');

To manage the class, you can simply select the class and then modify them. Look at this sample: 

                // Select universal class.
                $class('.header').set('display', 'none');
                
                $class('.header@desktop').set('width': '100%').on('hover', {background: '#ccc'});

You can also set multiple properties with this method: 

                // Select universal class.
                $class('.header').set({display: 'block', width: '100%', max_width: 960});

Tocm Media Query

Tocm Media Query is a function to register media query collections. $media(string name, string rules); Sample: 

                // Registering media query.
                $media('Desktop', 'all and (min-width: 1024px)');

                // Adding class to media query.
                $class('!.test@Desktop', {width: 100});
                $class('!.second-test@Desktop', {width: 100});

This is the css script generated by the sample above: 

                @media all and (min-width: 1024px) {
                    .test {
                        width: 100px;
                    }
                    .second-test {
                        width: 100px;
                    }
                }

Tocm Font

Tocm Font is a function to register web fonts. With Tocm Font you can add your font easily. $fonts(string name, array source, opt object property); Sample: 

                $fonts('Gotham Book', ['font/gotham.book.eot', 'font/gotham.book.ttf', 'font/gotham.book.woff']);

This is the css script generated by the sample above: 

                @font-face {
                    font-family: "Gotham Book";
                    src: local("Gotham Book");
                    src: url("font/gotham.book.eot");
                    src: url("font/gotham.book.eot?#iefix") format("embedded-opentype");
                    src: url("font/gotham.book.ttf") format("truetype");
                    src: url("font/gotham.book.woff") format("woff");
                }

Tocm Keyframe

Tocm Keyframe is a function to create a animation keyframes. $keyframes(string name, string position, object properties); Sample: 

                $keyframes('SlideLeft', '0%', {
                    left: '110%'
                })
                .at('50%', {
                    left: '50%'
                })
                .at('100%', {
                    left: '100%'
                });

This is the css script generated by the sample above: 

                @keyframes SlideLeft {
                    0% {
                        left: 110%;
                    }
                    50% {
                        left: 50%;
                    }
                    100% {
                        left: 100%;
                    }
                }
            
                @-webkit-keyframes SlideLeft {
                    0% {
                        left: 110%;
                    }
                    50% {
                        left: 50%;
                    }
                    100% {
                        left: 100%;
                    }
                }

Tocm Debugger

Tocm Debugger is a function to help you debug the layout with adding line on each box. $class.debugLayout(string lineColor); Sample: 

                // Debug with default line color.
                $class.debugLayout();
                // Debug with custom line color.
                $class.debugLayout('green');
                $class.debugLayout('#f33');
                // Debug with line color inheriting from the box color.
                $class.debugLayout('');
